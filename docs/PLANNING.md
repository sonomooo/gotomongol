# 고투몽골 (Go To Mongol) - 커스텀 투어 웹사이트 기획서

## 프로젝트 개요

| 항목 | 내용 |
|---|---|
| 서비스명 | **고투몽골 (Go To Mongol)** |
| 도메인 | **gotomongol.com** |
| 슬로건 | "나만의 몽골 여행을 설계하세요" |
| 컨셉 | 고객이 원하는 코스를 조합하면 견적을 산출해주는 맞춤형 프라이빗 투어 |
| 타겟 | 한국인 (20~40대, 자유여행 선호층) |
| 차별점 | 패키지 고정형 ❌ → 고객 선택 기반 커스텀 견적 |
| 운영 인원 | 3명 (가이드 1 + 현지관리 1 + 웹관리 1) |

## 도메인 구매 방법

### 추천: Cloudflare Registrar

1. https://dash.cloudflare.com 회원가입
2. 좌측 "Domain Registration" → "Register Domains"
3. `gotomongol.com` 검색
4. "Purchase" 클릭
5. 연락처 정보 입력 (WHOIS 자동 비공개 처리)
6. 카드 결제 (연 $10.44 ≈ 1.4만원)
7. 완료 → DNS 설정에서 서버 연결

> 전체 과정 10분 이내. Cloudflare는 마크업 0원 + CDN/SSL 무료 포함이라 가장 합리적.

## 사이트 구성

```
gotomongol.com
├── / (홈: 히어로 이미지 + "나만의 투어 만들기" CTA)
├── /custom (커스텀 투어 설계 폼 - 핵심 차별점)
│   └── 일수 → 스팟 → 액티비티 → 숙소 → 인원 → 견적 요청
├── /tours (추천 코스 목록)
│   └── /tours/{id} (상품 상세: 일정표, 가격, 포함사항)
├── /reviews (후기 게시판)
├── /guide (여행 가이드: 준비물, FAQ, 몽골 정보)
├── /about (회사 소개: 가족 소개, 현지 경험)
└── /admin (어드민: 견적 관리, 상품 관리)
```

## 커스텀 투어 설계 플로우

### 고객 선택 요소

| 단계 | 선택 항목 |
|---|---|
| ① 여행 일수 | 3일 / 5일 / 7일 / 10일+ |
| ② 관심 스팟 | 홉스골 호수, 고비사막, 테를지 국립공원, 오르혼 폭포, 쳉헤르 온천, 미니사막, 울란바토르 시내 |
| ③ 액티비티 | 승마, 낙타, 독수리 체험, 게르 숙박, 유목민 홈스테이, 별보기/은하수 촬영, 온천 |
| ④ 숙소 등급 | 캠프(기본) / 프리미엄 캠프 / 호텔 혼합 |
| ⑤ 인원 | 2인 / 4인 / 6인+ |

### 프로세스

```
고객: gotomongol.com/custom 에서 폼 작성 → 견적 요청
운영: 루트 설계 + 일정표 + 가격 산출 → 카톡/이메일 회신 (24시간 내)
고객: 확정 → 예약금 결제
```
